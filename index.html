Tags para textos

Voltar

Tag < p > cria um paragráfo.

Tag < u > sublinha o texto.

Tag < i > altera o texto para itálico.

Tag < b > altera o texto para negrito.

Use o atributo style="" dentro de uma tag para alterar cor, fonte e outras coisas no seu texto.

Tag para imagens

Voltar

Tag < img > insere uma imagem.

Use o atributo src="" dentro da tag para apontar o link da imagem.

Use o atributo width="" dentro da tag para ajustar o tamanho da imagem.
